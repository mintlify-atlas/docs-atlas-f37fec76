## Welcome to Workflow Engine Platform

Workflow Engine Platform is a complete async workflow engine designed for complex business processes that require human approval, multi-step processing, and reliable execution over days or weeks.

## Key Features

<AccordionGroup>
  <Accordion title="Visual Workflow Builder" icon="diagram-project">
    Create workflows visually with React Flow. Define steps, transitions, and approval policies through an intuitive drag-and-drop interface.
  </Accordion>
  
  <Accordion title="Human-in-the-Loop Processing" icon="user-check">
    Build workflows that pause for human approval, rejection, or revert actions. Configure assignees, priorities, and escalation policies.
  </Accordion>
  
  <Accordion title="Priority Chain & Multi-Assignee Support" icon="users">
    Notify users by priority order, all at once, or require approval from everyone. Escalate to the next assignee after timeout.
  </Accordion>
  
  <Accordion title="Subworkflow Mapping" icon="diagram-nested">
    Trigger child workflows from parent steps. The parent pauses until the child completes, then resumes with mapped outputs.
  </Accordion>
  
  <Accordion title="Real-time Notifications" icon="bell">
    Task inbox with in-app notifications. Users receive notifications only when workflows reach waiting steps.
  </Accordion>
  
  <Accordion title="Comprehensive Audit History" icon="clock-rotate-left">
    Every state change, action, and notification is persisted. View complete workflow history and action logs.
  </Accordion>
</AccordionGroup>

## Technology Stack

The platform is built on modern, reliable technologies:

- **Frontend**: Next.js 16 with shadcn/ui components
- **Backend**: FastAPI with Python
- **Database**: PostgreSQL (Neon) as the source of truth
- **Queue**: RabbitMQ for async task processing
- **Authentication**: Better Auth with email/password and JWT
- **Package Management**: pnpm for Node.js, uv for Python

## Use Cases

Workflow Engine Platform is ideal for:

- **Approval Workflows**: Invoice approval, purchase requisitions, expense reports
- **Review Processes**: Document review, code review, content moderation
- **Multi-Stage Operations**: Onboarding flows, release pipelines, compliance checks
- **Long-Running Processes**: Workflows that span days or weeks with multiple stakeholders
