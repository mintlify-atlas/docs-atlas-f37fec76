---
title: List My Notifications
api: GET /api/v1/me/notifications
---

Retrieves all notifications for the authenticated user. Notifications are returned in descending order by creation time (most recent first).

## Authentication

Requires Bearer token authentication. Returns only notifications for the authenticated user.

## Response

<ResponseField name="items" type="array">
  Array of notifications for the current user.
  
  <ResponseField name="items[].id" type="UUID">
    Unique identifier for the notification.
  </ResponseField>
  
  <ResponseField name="items[].workflowInstanceId" type="UUID | null">
    The workflow instance this notification is associated with (if applicable).
  </ResponseField>
  
  <ResponseField name="items[].stepInstanceId" type="UUID | null">
    The step instance this notification is associated with (if applicable).
  </ResponseField>
  
  <ResponseField name="items[].notificationType" type="string">
    Type of notification:
    - `task_assigned` - A task has been assigned to the user
    - `workflow_completed` - A workflow has been completed
    - `workflow_rejected` - A workflow has been rejected
    - `escalation` - A task has been escalated
    - `reminder` - A reminder for a pending task
  </ResponseField>
  
  <ResponseField name="items[].title" type="string">
    Notification title.
  </ResponseField>
  
  <ResponseField name="items[].body" type="string">
    Notification message body.
  </ResponseField>
  
  <ResponseField name="items[].isRead" type="boolean">
    Whether the notification has been read by the user.
  </ResponseField>
  
  <ResponseField name="items[].readAt" type="string | null">
    ISO 8601 timestamp when the notification was marked as read (if read).
  </ResponseField>
  
  <ResponseField name="items[].createdAt" type="string">
    ISO 8601 timestamp when the notification was created.
  </ResponseField>
</ResponseField>

## Example Request

<CodeGroup>

```bash cURL
curl -X GET https://api.example.com/api/v1/me/notifications \
  -H "Authorization: Bearer YOUR_TOKEN"
```

```javascript JavaScript
const response = await fetch('https://api.example.com/api/v1/me/notifications', {
  headers: {
    'Authorization': 'Bearer YOUR_TOKEN'
  }
});

const data = await response.json();
```

```python Python
import requests

response = requests.get(
    'https://api.example.com/api/v1/me/notifications',
    headers={'Authorization': 'Bearer YOUR_TOKEN'}
)

data = response.json()
```

## Use Cases

### Filter Unread Notifications

Get only unread notifications:

```javascript
const unreadNotifications = data.items.filter(notification => !notification.isRead);
```

### Filter by Type

Get all task assignment notifications:

```javascript
const taskNotifications = data.items.filter(notification => 
  notification.notificationType === 'task_assigned'
);
```

### Count Unread by Type

Count unread notifications by type:

```javascript
const unreadByType = data.items
  .filter(n => !n.isRead)
  .reduce((acc, n) => {
    acc[n.notificationType] = (acc[n.notificationType] || 0) + 1;
    return acc;
  }, {});
```

### Get Notifications for Specific Workflow

Filter notifications for a specific workflow instance:

```javascript
const workflowNotifications = data.items.filter(notification => 
  notification.workflowInstanceId === '123e4567-e89b-12d3-a456-426614174000'
);
```

### Display Badge Count

Show total unread count in UI:

```javascript
const unreadCount = data.items.filter(n => !n.isRead).length;

// Display in UI
if (unreadCount > 0) {
  badge.textContent = unreadCount > 99 ? '99+' : unreadCount;
  badge.style.display = 'block';
}
```

## Real-time Updates

Notifications are also delivered via [Server-Sent Events](/api/realtime/sse). Subscribe to the `notifications` channel to receive real-time notification updates:

```javascript
const eventSource = new EventSource(
  'https://api.example.com/api/v1/events',
  {
    headers: {
      'Authorization': 'Bearer YOUR_TOKEN'
    }
  }
);

eventSource.addEventListener('realtime', (event) => {
  const payload = JSON.parse(event.data);
  
  if (payload.type === 'runtime.notifications.changed') {
    // Refresh notifications list
    fetchNotifications();
  }
});
```
